Opzoekwerk Microsoft Bot Framework

Posted by Maxime De Grauwe on December 15, 2019

Opzoekwerk Microsoft Bot Framework

Deze weekend heb ik me bezig gehouden met het opzoeken van informatie over de Microsoft Bot Framework. Het bedrijf (Arinti) met wie we ons Final Work doen (mentor) gebruikt ook dat framework. Aangezien er beslist werd dat ik een chatbot ging maken met AI (Nicolas besloot zijn project anders aan te pakken), besloten we dat ik Microsoft Bot Framework ging gebruiken. Het kan dat ik eventuele moeilijkheden communiceer aan Arinti, in de hoop dat ze de tijd zullen hebben om mij te kunnen helpen. Ze zouden ook tips kunnen geven over hoe ik het project kan aanpakken.

In mijn verslag heb ik uitvoerig uitleg gegeven van wat ik gelezen heb. Natuurlijk begon ik eerst te schrijven over wat ik geleerd heb over Microsoft Bot Framework zelf. Ik begreep dat , in mijn geval ik LUIS (Language Understanding Intelligent Service) zou moeten leren gebruiken. Het is een machine learning-based service die helpt om een gesprek zo natuurlijk mogelijk te maken. Dus daarrond heb ik ook wat informatie gaan zoeken.

Vervolgens heb ik wat reviews geraadpleegd over de Microsoft Bot Framework en ook verschillen gaan zoeken tussen andere frameworks die chatbots maken. In het algemeen is de Microsoft Bot Framework geapprecieerd en komt vaak in de "top 10 bot frameworks to use". Wat ik eruit kon begrijpen, is dat de framework krachtig kan worden omdat Microsoft andere tools ter beschikking stelt aan developers. Dankzij de framework zijn al die services gekoppeld aan elkaar en kunnen ze dus samenwerken.

Ondertussen ben ik al begonnen aan een tutorial om een chatbot te maken en die te deployen op Azure. Ik ben nog niet klaar hiermee, maar ik werk eraan. Is het mogelijk om daaruit een prototype te maken voor het echte werk begint? Het zal vooral interessant worden wanneer ik de chatbot zal beginnen in te stellen om hem te laten werken zoals ik het wens. Dat zal ik nog moeten zien.
